# 📘 Student Notes Management System

## 📌 Project Overview

This is a full-stack web application that allows students to manage their notes efficiently. Users can register, login, and perform CRUD operations (Create, Read, Update, Delete) on notes.

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 📝 Add, Edit, Delete Notes
* 📊 Dashboard with Analytics
* 🎨 Responsive UI using Tailwind CSS
* 🌐 Full-stack (React + Node.js + MongoDB)

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

---

## 📂 Project Structure

```
project/
│── client/    # Frontend (React)
│── server/    # Backend (Node.js)
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/student-notes-app.git
```

### 2. Backend setup

```
cd server
npm install
node server.js
```

### 3. Frontend setup

```
cd client
npm install
npm run dev
```
